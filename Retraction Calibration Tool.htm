Настройки принтера


Настройки по умолчанию должны работать с боуденом.
Сначала попробуйте использовать значения по умолчанию.
Затем отрегулируйте температуру вентилятора и т.д.
Записи: Для delta установите размер кровати равным 0 для x и y
Для прямого привода начните с .5 начального втягивания и .25 приращений
Для Боудена начните с .5 начального отвода и .5 приращения
